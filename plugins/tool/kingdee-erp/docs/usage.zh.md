# 星智 使用指南

本指南面向日常使用者，帮助你快速上手星智，用自然语言查询和操作金蝶云星空 ERP 系统。

---

## 产品介绍

星智是一个 AI 驱动的 ERP 智能助理。它连接金蝶云星空 WebAPI，让你用日常对话的方式完成 ERP 操作，无需记住菜单路径或字段编码。

核心能力：

- **自然语言查询**：用日常用语提问，助手自动翻译为金蝶 API 调用并返回结果
- **5 个专业角色**：财务、销售、库存、采购、高管决策，各司其职，语义路由自动分发
- **18 项 ERP 工具**：覆盖查询、新增、删除、提交、审核、反审核、下推、工作流审批等全量操作
- **写入安全防护**：所有写入操作必须经过用户确认才能执行，防止误操作
- **数据权限隔离**：每个用户只能看到自己有权限的组织和业务域数据

## 快速开始

### 打开助手

在浏览器中访问系统地址（由管理员提供，默认格式 `http://<服务器IP>:8088`），进入控制台。

### 选择角色

控制台左侧导航栏有 5 个专业 Agent。你可以直接和默认 Agent 对话，它会根据你的问题自动路由到对应的专业角色。也可以手动切换到特定角色。

| 角色 | 擅长领域 | 典型问题 |
|------|----------|----------|
| 财务助手 | 总账、应收、应付、固定资产、财务报表 | "查一下本月应收账款余额" |
| 销售助手 | 销售订单、客户、报价、出库、发货 | "最近一周有哪些未发货的销售订单？" |
| 采购助手 | 采购订单、供应商、入库、比价 | "帮我查供应商'东方钢铁'的采购订单" |
| 库存助手 | 出入库、盘点、库存查询、调拨 | "A4打印纸当前库存多少？" |
| 高管决策助手 | 跨域汇总、经营分析、多系统对账 | "本季度各产品线销售额对比" |

### 开始对话

在底部输入框直接输入问题，按回车发送。助手会自动调用金蝶 ERP 数据回答你。

## 对话指南

### 用自然语言提问

不需要记住金蝶的 FormId、字段名或操作路径。直接说人话：

好的问法：
- "帮我查一下客户'华星科技'的基本信息"
- "上个月销售金额最高的前10个客户是谁？"
- "帮我新增一张采购订单，供应商选东方钢铁"

不推荐的问法：
- "执行 ExecuteBillQuery，FormId=BD_Customer，字段 FName,FNumber"

助手在后台会自动完成表单搜索、字段映射、API 调用等步骤。

### 查询操作

查询是只读操作，不需要确认。直接提问即可：

- 精确查询："查订单号 SO-2024-0015 的详情"
- 模糊查询："最近一周有哪些超过5万的销售订单？"
- 报表查询："帮我查利润表"
- 合并报表："查一下2025年第一季度的合并报表"

### 写入操作

所有写入操作（新增、删除、提交、审核、反审核、下推、工作流审批）都采用**双步确认**机制：

1. **第一步**：助手展示操作预览，告诉你将要做什么、影响哪些数据
2. **第二步**：你确认无误后，助手才执行实际操作

示例流程：

```
你：帮我新增一张采购订单，供应商东方钢铁，物料冷轧钢板，100吨

助手：
  ⚠️ 写入操作预览（尚未执行）
  📋 操作类型：保存/新增单据
  📄 目标表单：PUR_PurchaseOrder
  🏢 组织ID：01
  📝 操作详情：
    • 供应商：S001（东方钢铁）
    • 物料：M00123（冷轧钢板）
    • 数量：100
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  如确认执行，请再次调用本工具并设置 execute=True

你：确认执行

助手：保存成功，单据编号 PO-2026-0088
```

**你始终拥有最终决定权。未经你确认的操作不会被执行。**

### 多步操作

复杂操作可以分步完成。比如创建采购订单时，助手会自动：

1. 查询供应商编号（确保编码准确）
2. 查询物料编号
3. 组装单据数据
4. 展示预览
5. 等待你确认
6. 执行保存

你不需要手动执行这些步骤，只需用自然语言描述需求。

### 产品智能问答

除了查数据，助手还能解答金蝶产品的使用问题：

- "金蝶怎么设置审批流？"
- "销售订单下推出库单报错怎么办？"
- "如何配置多币种核算？"

助手会从金蝶知识库中搜索答案。

## 专业角色

### 角色说明

| 角色 ID | 名称 | 业务域 | 说明 |
|---------|------|--------|------|
| erp-finance | 财务助手 | 总账(GL)、应收(AR)、应付(AP)、固定资产 | 凭证、科目、余额表、利润表、资产负债表 |
| erp-sales | 销售助手 | 销售(SAL) | 销售订单、客户、报价、出库、发货 |
| erp-purchase | 采购助手 | 采购(PUR) | 采购订单、供应商、入库、比价 |
| erp-inventory | 库存助手 | 库存(STK) | 出入库、盘点、库存查询、调拨 |
| erp-executive | 高管决策助手 | 全域 | 跨域汇总、经营分析、多系统对账、报表摘要 |

### 语义路由

默认 Agent 充当路由器。当你提问时，它会分析问题涉及的业务域，自动转发给对应的专业角色。如果问题不涉及 ERP，它会直接回答。

路由规则：

| 问题关键词 | 路由目标 |
|-----------|----------|
| 凭证、科目、总账、应收、应付、余额表、利润表、资产负债表 | 财务助手 |
| 销售订单、客户、报价、出库、发货 | 销售助手 |
| 采购订单、供应商、入库、比价 | 采购助手 |
| 库存、盘点、出入库、仓库、物料 | 库存助手 |
| 经营分析、汇总、对比、对账、决策 | 高管决策助手 |
| ERP 相关但不确定具体域 | 财务助手（默认） |
| 非 ERP 问题 | 默认 Agent 直接回答 |

| 工具 | 说明 |
|------|------|

## 权限与安全

### 你能看到什么

你的数据访问范围由管理员分配的权限决定，包含三个维度：

- **组织范围**：你能访问哪些金蝶组织的数据
- **业务域**：你能访问哪些模块（财务、销售、采购、库存等）
- **访问级别**：只读（仅查询）或读写（可执行写入操作）

### 权限不足时

如果你尝试访问没有权限的数据，助手会提示：

```
您没有组织 02 的查询权限。
您有权限的组织: 01
请联系管理员配置您的权限。
```

提示中会附带管理员联系方式（如果管理员已配置）。

### 写入确认流程

所有写入操作都会先展示预览：

```
⚠️ 写入操作预览（尚未执行）
📋 操作类型：保存/新增单据
📄 目标表单：PUR_PurchaseOrder
🏢 组织ID：01
📝 操作详情：
  • 供应商：S001
  • 物料：M00123
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
如确认执行，请再次调用本工具并设置 execute=True
```

你可以：
- 确认执行：回复"确认"或"执行"
- 修改参数：告诉助手需要调整什么
- 取消操作：说"取消"或忽略

### 删除操作特别提醒

删除操作是不可逆的。预览时会显示警告信息：

```
🚨 删除操作不可逆！删除后数据无法恢复！
```

请仔细核对单据编号后再确认。

## 常见问题

### 助手说"没有权限查询XXX"

你当前账号没有该数据的访问权限。请联系管理员，在权限管理中为你分配对应的组织和业务域权限。

### 助手返回的数据和金蝶客户端不一致

可能原因：
- 权限限制导致部分字段被过滤
- 查询条件不同（如时间范围、组织范围）
- 金蝶数据在查询后发生了变化

建议：让助手用更精确的条件重新查询。

### 助手说"查询结果为空"

金蝶系统中确实没有匹配的数据。请核实：
- 查询条件是否正确（日期范围、单据状态等）
- 组织是否正确（是否查错了组织）
- 数据是否已录入金蝶系统

助手不会编造数据。如果查询结果为空，它会如实告知。

### 助手无法理解我的问题

尝试：
- 用更具体的描述（如指定时间范围、单据类型）
- 分步提问（先查询，再操作）
- 换一种说法

### 如何查看历史对话

在控制台左侧导航的"会话"页面可以查看所有历史对话记录。

### 可以同时和多个角色对话吗

可以。在控制台中你可以创建多个会话，每个会话可以和不同的角色对话。

### 写入操作失败了怎么办

常见原因：
- 单据状态不允许操作（如已审核的单据不能删除）
- 必填字段缺失
- 金蝶业务规则校验失败

助手会返回金蝶的错误信息，请根据提示调整后重试。如果问题持续，联系管理员排查。
